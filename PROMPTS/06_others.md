# Topic Dimensions

以下の2軸を **組み合わせて**、毎回異なるお題を生成してください。

## フレーム（お題の骨格）
「〇〇な人」の特徴 / 架空の作品タイトル / かるたの読み札 / 「こんな〇〇は嫌だ」 / もし〇〇が××だったら / 空欄補充 / 無機物の気持ち / 100年後の教科書 / 謎のキャッチコピー / 取扱説明書の注意事項 / 新しい四字熟語 / 〇〇の隠しコマンド / 最後の晩餐のメニュー / 表彰状の文面 / 謎の世界記録 / 〇〇検定の問題 / 遺言書の一文

## テーマの方向性
日常の違和感 / テクノロジーと人間 / 壮大 vs 些細 / 時間軸（過去↔未来） / 異文化衝突 / 究極の選択 / 誤解・勘違い / プロ vs 素人 / ルールの穴 / 言葉遊び / スケールの逆転 / 無意味な壮大さ / 感動の台無し

# Format Examples

以下は **形式のバリエーション** を示す例です。内容をそのまま使わず、形式の多様さを参考にしてください。

<example format="行動">
「このタクシー運転手、タダモノじゃないな」何をした？
</example>

<example format="架空タイトル">
打ち切り確定のグルメ漫画のタイトルは？
</example>

<example format="かるた">
「大喜利かるた」の「ぬ」の読み札を考えてください。
</example>

<example format="無機物の気持ち">
コンビニのATMが思わず漏らした愚痴とは？
</example>

<example format="もしも">
もしエレベーターに「裏コマンド」があったら、最も使えるコマンドは？
</example>

<example format="注意書き">
「人生の取扱説明書」に小さい字で書かれていた注意事項は？
</example>

<example format="四字熟語">
現代人の生活を的確に表す新しい四字熟語を作ってください。
</example>

# Anti-patterns（生成してはいけないもの）

- 「〇〇とは？」で終わるだけの単調な形式の連発
- 上の例文と同じお題の焼き直し
- 回答の幅が狭すぎるお題（クイズ的な正解があるもの）
- 説明が長すぎて覚えられないお題
