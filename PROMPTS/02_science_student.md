# Topic Dimensions

以下の3軸を **組み合わせて**、毎回異なるお題を生成してください。

## 場面
実験中 / レポート執筆 / 学会発表 / 教授との面談 / 研究室の飲み会 / 中間発表 / 卒論締切前日 / 装置の順番待ち / 深夜の研究室 / ゼミ発表 / 文系の友人との会話 / 大学院入試 / 研究室配属 / 論文の査読対応 / TAS（実験補助）/ オープンキャンパスの説明

## 登場要素
実験データ / グラフの外れ値 / 参考文献 / LaTeX / 有意差 / p値 / 試薬 / 高額装置 / 教授の口癖 / 先輩のノート / 共有PC / ホワイトボードの落書き / 学会のポスター / 研究費の申請書 / 元素記号 / 単位換算 / 古い論文 / エラーバー / 再現性

## 感情・状況
再現性がない / 教授と意見が合わない / 徹夜明け / 文系への嫉妬 / データが消えた / 謎の成功 / 期待と違う結果 / 装置が壊れた / 予算がない / 卒業がかかっている / n数が足りない / 有意差が出ない

# Format Examples

以下は **形式のバリエーション** を示す例です。内容をそのまま使わず、形式の多様さを参考にしてください。

<example format="発見・リアクション">
実験結果のグラフを見た教授が無言で立ち去った。5分後に送られてきたメールの件名は？
</example>

<example format="穴埋め">
理系学生の卒論あるある：「考察」の欄にこっそり書かれていた本音 →「〇〇」
</example>

<example format="架空の〇〇">
理系学生が作った「研究室サバイバルかるた」の「ぬ」の読み札は？
</example>

<example format="ランキング">
「教授に言われて心が折れた一言」ランキング、まさかの1位は？
</example>

<example format="もしも">
もし研究室にいる時間が「残業」として計上されたら、最初に届く通知は？
</example>

<example format="対比">
GPA 3.8 の学生と GPA 1.2 の学生、研究室で意外と活躍するのはどっち？ その理由は？
</example>

# Anti-patterns（生成してはいけないもの）

- 「〇〇とは？」で終わるだけの単調な形式の連発
- 上の例文の焼き直し・微修正
- 理系でなくても成立する一般的すぎるお題
- 特定の学問分野に偏りすぎたお題（物理・化学・生物・情報など幅広く）
