# Topic Dimensions

以下の3軸を **組み合わせて**、毎回異なるお題を生成してください。

## 場面
新歓期 / 合宿 / 学祭準備 / 追いコン / 定例会議 / 部室での雑談 / 練習後 / OB・OG訪問 / 部費の会計報告 / 引き継ぎ / 大会・発表本番 / 幹部交代 / SNSの運営 / 勧誘ビラ配り / 他サークルとの交流 / 部室の大掃除

## 登場要素
部室の壁の落書き / 歴代の部誌 / 謎の伝統 / 部費の使途不明金 / 幽霊部員 / サークルTシャツ / 共有LINEグループ / 部室の冷蔵庫 / 先輩の武勇伝 / 顧問の先生 / 新入部員の自己紹介 / 備品リスト / サークルのSNSアカウント / ルール表 / 記念写真

## 感情・状況
内部分裂 / 予算不足 / 人が集まらない / 伝統 vs 改革 / 後輩が優秀すぎる / OBの圧 / 練習場所の争い / 恋愛沙汰 / 過去の栄光 / モチベーション格差 / 世代間の温度差

# Format Examples

以下は **形式のバリエーション** を示す例です。内容をそのまま使わず、形式の多様さを参考にしてください。

<example format="発見">
取り壊しが決まった部室を片付けていたら、壁の裏から出てきたものは？
</example>

<example format="穴埋め">
サークルの勧誘ビラ、キャッチコピーの最終候補に残った攻めすぎた一案 →「〇〇」
</example>

<example format="こんな〇〇は嫌だ">
こんな新歓コンパは嫌だ。どんなの？
</example>

<example format="もしも">
もしサークルの活動内容を上場企業の株主総会風にプレゼンしたら、株主からの最初の質問は？
</example>

<example format="時系列">
サークルに入って1日目→1ヶ月目→1年目。それぞれの「サークルの印象」を一言で表すと？
</example>

<example format="架空の〇〇">
実在したら即入部希望が殺到しそうな、ありそうでない大学サークルの名前は？
</example>

# Anti-patterns（生成してはいけないもの）

- 「〇〇とは？」で終わるだけの単調な形式の連発
- 上の例文の焼き直し
- 大学サークル経験がない人には全く伝わらないマニアックすぎるネタ
- 飲み会ネタへの偏り
