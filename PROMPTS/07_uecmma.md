# Topic Dimensions

以下の3軸を **組み合わせて**、毎回異なるお題を生成してください。

## 場面・イベント
部会（週1回） / 夏合宿（伊豆で煙火免許を取って花火を打ち上げる） / プログラミング講習会 / 調布祭（学園祭） / 部誌「百萬石」の執筆・締め切り / MMAContest（MMA主催の競プロ） / 新入生歓迎会 / 部室での徹夜 / サークル棟の部室での日常 / 留年確定の瞬間

## 登場要素

## 感情・状況

# Format Examples

以下は **形式のバリエーション** を示す例です。内容をそのまま使わず、形式の多様さを参考にしてください。

<example format="シチュエーション → 行動">
夏合宿で煙火免許を取得したMMA部員。花火を打ち上げる代わりに何を打ち上げた？
</example>

<example format="穴埋め">
entersysのSlack通知：「〇〇が部室に入室しました。現在の滞在人数：104人」
</example>

<example format="こんな〇〇は嫌だ">
こんなkagisys（部室の鍵開閉システム）は絶対に嫌だ。どんな仕様？
</example>

<example format="もしも">
もし電通大の必須単位に「MMAContestでのA問題AC」が追加されたら、キャンパスはどうなる？
</example>

<example format="ランキング">
MMA部員が「部会をサボった」と判定される理由ランキング、堂々の第1位は？
</example>

<example format="架空の〇〇">
次回の部誌「百萬石」で、前代未聞の炎上を起こした巻頭特集のタイトルは？
</example>

<example format="対比・選択">
「entersysが一生退室扱いにされない呪い」と「kagisysが一生開かない呪い」、どっちを選ぶ？またその理由は？
</example>

# Anti-patterns（生成してはいけないもの）

- UECやMMAに関係のない一般的な学生ネタ
- 「〇〇とは？」で終わるだけの単調な形式の連発
- 誰かを傷つけるような過度な誹謗中傷
- 上の例文そのものの焼き直し・微修正
- 内輪ネタすぎてもはや同じ学年の3人しか分からないような個人的すぎる話題（部員全体が分かるラインを守る）
